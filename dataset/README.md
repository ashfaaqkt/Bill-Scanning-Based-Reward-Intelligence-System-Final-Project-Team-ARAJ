## Folder Structure

```
dataset/
  genuine/          ← CORD + SROIE images go here (gitignored, download from Drive)
  indian/           ← Jyoti's 50 raw receipt photos go here (gitignored, download from Drive)
  tampered/         ← Generated by generate_tampered.py (gitignored)
  test_bills/       ← 5 sample bills for OCR smoke testing (in repo)
  processed/
    labels.csv      ← 151 labelled samples (Ranjeet 101 + Jyoti 50)
    receipts_master.csv ← Full feature CSV for ML training (populate after downloading images)
  generate_tampered.py   ← Creates tampered variants (4 modes: brightness, duplicate_copy, number_overwrite, handwritten)
  exact_duplicate_check.py ← Flags exact duplicate receipts from CSV
  perceptual_hash.py     ← pHash image similarity comparison tool
  fraud_detector.py      ← Standalone multi-signal fraud scorer
```

## processed/receipts_master.csv Schema

```
image_path, merchant, date, total, category, items_text, source
```

## processed/labels.csv Schema

```
image_path, label, labelled_by, notes
```

Labels: `genuine` / `tampered` / `blurry` / `multi_bill` / `handwritten`
